# VAIBHAV HADE - Personal Portfolio Website

A modern, responsive portfolio website with a dark techy theme and futuristic AI vibes, built with HTML, CSS, and JavaScript.

## Features

- Dark techy theme with futuristic AI aesthetics
- Fully responsive design (mobile & desktop)
- Smooth scrolling navigation
- Interactive elements with hover effects
- Glitch text animation for the hero section
- Animated skill tags and project cards
- Particle background effects
- Social media integration

## Sections

1. **Home** - Introduction with name and title
2. **About** - Educational background and personal description
3. **Skills** - Technical skills organized by category
4. **Projects** - Main projects and extra projects
5. **Contact** - Social media links

## Deployment to GitHub Pages

### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and log in to your account
2. Click the "+" icon in the upper right corner and select "New repository"
3. Name your repository `Vaibhav2422.github.io`
4. Make the repository public
5. Do NOT initialize with a README
6. Click "Create repository"

### Step 2: Push Your Code to GitHub

In your project directory, run these commands in your terminal or command prompt:

```bash
git init
git add .
git commit -m "Initial commit: Portfolio website"
git branch -M main
git remote add origin https://github.com/Vaibhav2422/Vaibhav2422.github.io.git
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. On your repository page, click "Settings" tab
2. Scroll down to the "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click "Save"
6. Wait a few minutes for GitHub to deploy your site

### Step 4: View Your Live Website

After deployment, your website will be available at:
`https://your-username.github.io`

It may take a few minutes for your site to be published. Refresh the page after 1-2 minutes.

## Custom Domain (Optional)

If you want to use a custom domain:

1. In your repository settings, go to the "Pages" section
2. Under "Custom domain", enter your domain name
3. Follow GitHub's instructions to configure your DNS settings with your domain provider

## Local Development

To view your website locally:

1. Open `index.html` in your web browser, OR
2. Use a local server (recommended):
   - If you have Python installed: Run `python -m http.server 8000` in the project directory
   - Then visit `http://localhost:8000` in your browser

## Customization

To customize this portfolio for your own use:

1. Edit `index.html` to update personal information
2. Modify `styles.css` to change colors, fonts, or layout
3. Update `script.js` for any interactive behaviors
4. Replace social media links in the contact section
